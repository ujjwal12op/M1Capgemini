+91-
This is just a string literal.

Every contact number will start with +91- (India country code).

2. ||
In Oracle, || means string concatenation (joins two strings together).

3. SUBSTR(cm.CONTACT_NO, 1, 3)
SUBSTR(string, start_position, length)

This takes the first 3 digits of CONTACT_NO.
Example: If contact is 8764562345, this returns 876.

4. '-' || SUBSTR(cm.CONTACT_NO, 4, 3)
First, '-' adds a dash.

SUBSTR(cm.CONTACT_NO, 4, 3) takes 3 digits starting at position 4.
From 8764562345, starting at digit 4 gives 456.

5. '-' || SUBSTR(cm.CONTACT_NO, 7, 4)
Again, adds a dash.

Takes 4 digits starting at position 7.
From 8764562345, starting at digit 7 gives 2345.
