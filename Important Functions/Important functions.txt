Date / Time Functions
	EXTRACT(YEAR/MONTH FROM date) → Extract year.
	TO_DATE('05:00:01', 'HH24:MI:SS') → Convert string to date with time.
	CURRENT_TIME + INTERVAL '1' HOUR → Add 1 hour to current time.
	CURRENT_TIME + INTERVAL '5' DAYS → Add 5 days to current time.
	TO_CHAR(Date, 'FMMonth') → Month name from date.
	DATEADD(YEAR,-2,GETDATE()) : 2 YEARS BEFORE.
	CURRENT_DATE → Current date in session’s timezone.
	ADD_MONTHS(date, n) → Add n months to a date.
	MONTHS_BETWEEN(date1, date2) → Months difference.
	EXTRACT(DAY FROM date) → Extract day.
	EXTRACT(HOUR FROM timestamp) → Extract hour.
	TO_TIMESTAMP('2024-08-12 05:00:01', 'YYYY-MM-DD HH24:MI:SS') → Convert string to timestamp.
________________________________________
2. String Functions
	SUBSTR(String, 1, 3) → Extract substring from position 1, length 3.
	SUBSTR(String, 4, 3) → Extract substring from position 4, length 3.
	CONCAT(String1, '&', String2) → Concatenate with ampersand.
	|| → Concatenate strings.
	UPPER(string) → Convert to uppercase.
	LOWER(string) → Convert to lowercase.
	LEFT(String,1)-> takes first value of string from left.
	RIGHT(String,1) -> Takes right value of string.
	INITCAP(string) → Capitalize first letter of each word.
	REPLACE(string, 'old', 'new') → Replace substring.
	LENGTH(string) → Get string length.
________________________________________
3. Numeric Functions
	ROUND(number) → Round to nearest integer.
	TO_CHAR(Salary, 'FM999,999,999') → Format number with commas.
	ABS(num) → Absolute value.
	CEIL(num) → Ceiling value.
	FLOOR(num) → Floor value.
	MOD(m, n) → Remainder.
	POWER(m, n) → m raised to n.
	SQRT(num) → Square root.
	TRUNC(num, decimals) → Truncate decimals.
	ROUND(num, decimals) → Round with decimals.
________________________________________
4. Conditional / Null Handling Functions
	COALESCE(COUNT(movie_id), 0) → Replace null with 0.
	NVL(val1, val2) → Replace null with val2.
	CASE WHEN condition THEN val1 ELSE val2 END → Conditional logic.
	NVL2(expr1, expr2, expr3) → expr2 if expr1 not null, else expr3.
	COALESCE(expr1, expr2, ...) → First non-null value.
	NULLIF(expr1, expr2) → Null if expr1 = expr2.
________________________________________
5. Aggregate Functions
	SUM(column) → Total sum.
	AVG(column) → Average.
	MIN(column) → Minimum.
	MAX(column) → Maximum.
	COUNT(column) → Count non-null values.
	COUNT(*) → Count all rows.
________________________________________
6. Analytical (Window) Functions (sometimes in M1 level)
	ROW_NUMBER() OVER (ORDER BY col) → Sequential numbering.
	RANK() OVER (ORDER BY col) → Rank with gaps.
	DENSE_RANK() OVER (ORDER BY col) → Rank without gaps.
	LEAD(col, n) → Value from next row.
	LAG(col, n) → Value from previous row.
